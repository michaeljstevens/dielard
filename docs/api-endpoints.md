# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

* `GET /api/users`
* `GET /api/users/:id`
* `POST /api/users`
* `PATCH /api/users/:id`

### Session

* `POST /api/session`
* `DELETE /api/session`
* `GET /api/session`

### Routes

* `POST /api/routes`
* `DELETE /api/routes/:id`
* `GET /api/routes`
* `GET /api/routes/:id`
